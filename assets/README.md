# Image asset slots

このゲームは、以下の透過PNGを置くと自動で表示します。未配置の間はCSSと絵文字の仮表示になります。

- `assets/treasure-closed.png`
- `assets/treasure-open.png`
- `assets/items/{dan}-{mode}.png`
- `assets/outfits/{dan}-full.png`
- `assets/outfits/review-1-3-full.png`
- `assets/outfits/review-4-6-full.png`
- `assets/outfits/review-7-9-full.png`
- `assets/outfits/review-all-full.png`
- `assets/outfits/summary-1-3-full.png`
- `assets/outfits/summary-4-6-full.png`
- `assets/outfits/summary-7-9-full.png`
- `assets/outfits/summary-all-full.png`

`{dan}` は `1` から `9`、`{mode}` は `learn`、`choice`、`sequence`、`random`、`master` です。
